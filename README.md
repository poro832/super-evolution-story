# 슈퍼 진화 스토리 공략

모바일 게임 **슈퍼 진화 스토리(슈진스)** 비공식 팬 공략 사이트 (개인 취미용).

순수 정적 사이트(HTML/CSS/JS)이며 GitHub Pages로 배포합니다.

🔗 **배포 주소:** https://poro832.github.io/super-evolution-story/

## 페이지
- **종합 공략** — 태초·신생 육성, 컨텐츠 진행 팁, 제단·상점·젬
- **덱 공략** — 희전완포 메타 현역 티어덱 (접이식)
- **신생** — 티어표, 공통 세팅, 신생별 상세 세팅 (이미지 클릭 모달)
- **입문 가이드 / 수호신 / 룬·유전자**

## 자료 출처
슈진스 네이버 카페 자료 및 나무위키 「슈퍼 진화 스토리」를 참고해 재구성했습니다.
